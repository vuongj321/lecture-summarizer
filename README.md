# Monkey Mentor 🐒

🌐 [monkeymentor.app](https://monkeymentor.app)

An AI-powered study tool that turns lecture PDFs into summaries, flashcards, and an AI tutor in seconds. Upload any lecture, get a clean structured breakdown of the key concepts, ask questions about your material in a chat interface, and study with a flashcard deck built from your actual notes. No generic content, no re-reading 80 slides the night before an exam.

## Previously LectureAI 📚

Monkey Mentor started as **LectureAI**, a project built for the Amazon Nova Hackathon. The original goal was simple: upload a lecture PDF and get an AI-generated summary instantly.

After the hackathon, LectureAI was rebuilt from the ground up into a production-grade application. The stack was hardened with JWT authentication, S3 file storage, SSE streaming, and a full CI/CD pipeline. That foundation became Monkey Mentor, a full study platform with chat, flashcards, quizzes, and progress tracking, built on top of what LectureAI started.
